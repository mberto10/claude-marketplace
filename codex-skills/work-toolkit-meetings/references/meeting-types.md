# Meeting Types

## Jour Fixe (JF)
- Goal: Progress review, blockers, decisions, next steps
- Inputs: Linear activity + YouTrack epic status
- Output: Agenda with time boxes and decision points

## Stakeholder Update
- Goal: Outcomes, timeline, risks, decisions needed
- Inputs: Milestones, latest KW update, blockers
- Output: Executive summary + highlights + risks

## Team Sync
- Goal: Technical progress, dependencies, blockers
- Inputs: Linear tasks, PRs, tech decisions
- Output: Work-in-progress table + action items

## Lenkungsausschuss
- Goal: Cross-project status + decisions
- Inputs: KW updates for all projects
- Output: Short summaries per project and decisions
