# Email Patterns (German)

## Status Update (Standard)

```markdown
Betreff: Status Update: [Projektname] - KW [XX]

Hallo zusammen,

hier ein kurzes Update zum aktuellen Stand von [Projektname]:

**Erledigt diese Woche:**
- [Aufgabe 1]
- [Aufgabe 2]

**In Arbeit:**
- [Aufgabe 3] (ETA: [Datum])

**Blocker/Risiken:**
- [Falls vorhanden]

**Naechste Schritte:**
- [Schritt 1]
- [Schritt 2]

Bei Fragen meldet euch gerne.

Beste Gruesse
[Name]
```

## Quick Update (Kurz)

```markdown
Betreff: Quick Update: [Thema]

Hi [Name],

kurzes Update: [1-2 Saetze zum Stand]

Naechster Schritt meinerseits: [Aktion]

VG
[Name]
```

## Meeting Follow-Up

```markdown
Betreff: Follow-Up: [Meeting-Titel] vom [Datum]

Hallo zusammen,

danke fuer das produktive Meeting. Hier die wichtigsten Punkte:

**Besprochene Themen:**
- [Thema 1]
- [Thema 2]

**Entscheidungen:**
- [Entscheidung 1]

**Action Items:**
- [ ] [Person]: [Aufgabe] bis [Datum]

Beste Gruesse
[Name]
```
