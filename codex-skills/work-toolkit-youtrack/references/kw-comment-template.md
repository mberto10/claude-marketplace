# KW Comment Template

```markdown
## KW{XX}

**Updates:**
- [Update 1]
- [Update 2]

**Blocker:**
- [Blocker]   # or "Keine"

**Next Steps:**
- [Next step 1]
- [Next step 2]
```

## Example

```markdown
## KW50

**Updates:**
- System Prompt v2 deployed
- 60+ Testfragen verarbeitet

**Blocker:**
- Keine

**Next Steps:**
- Weiteres Testing mit Customer Support
- Knowledge Base Refresh bei Bedarf
```
